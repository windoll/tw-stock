# 台股即時報價

單一 HTML 的台股自選股工具：即時報價、持股損益、股利追蹤，資料存在瀏覽器，免後端、可直接放 GitHub Pages。

## 功能

### 報價
- 輸入**代號或名稱**加入自選（例：`2330` 或 `台積電`），含自動完成下拉。
- 即時報價：成交、漲跌、幅度、開高低、昨收、量（資料約有數秒延遲，非毫秒級）。
- 表格／卡片兩種檢視，可拖曳調整排序。
- 頂部顯示**加權指數**與**櫃買指數**。
- 自動定時更新（10／30／60 秒或關閉）；分頁切走時自動暫停，切回來恢復。
- 欄位可自訂（「欄位」按鈕勾選顯示哪些欄）；手機自動精簡顯示。

### 持股與損益
- 每檔可記**交易紀錄**（多筆買／賣，含日期、股數、價格）：點該檔的 ✎。
- 自動計算**加權平均成本**與目前持股。
- 顯示未實現損益、報酬率、**含息報酬**（價差＋實際領到的現金股利）。
- 現金**殖利率**（近一年現金股利 ÷ 現價）。
- **投資組合佔比圖**（各持股市值佔比）。
- 上方總計：總成本、總市值、總損益、含息損益。

### 股利（獨立頁簽，手動同步）
- 按「同步股利」抓取你清單內股票的股利（含 **ETF 與個股**、有歷史）。
- 採**真正的除權息日**；現金、股票股利分開逐筆列示，可展開／收合。
- 顯示除權息日、**實際發放日**、每股配發（小數三位）。
- 可輸入**日期區間**，計算期間內領到的現金／股票股利（依「各除息日當下實際持股」精算）。
- **各年度現金股利**長條圖（依當年實際持股）。
- **即將除息提醒**（清單內 14 天內除權息）。

### 個股走勢圖
- 每檔旁的 📈，看近 3 個月收盤折線。

### 其他
- 👁 一鍵模糊隱藏所有獲利／持股／股利金額。
- **匯出／匯入**：一鍵備份／還原自選與交易紀錄（JSON）。

## 使用

直接用瀏覽器開啟 `index.html` 即可。

部署到 GitHub Pages：repo → Settings → Pages → Source 選 `main` / root，
之後可由 `https://你的帳號.github.io/tw-stock/` 開啟。

> 提示：資料只存在你的瀏覽器（localStorage），換電腦或清快取會消失，請定期用「匯出」備份。

## 定期定額怎麼填

在該檔的 ✎ 新增一筆「買」，填總股數、平均成本，日期填「大約開始扣款的年月」即可。
（日期留空會被當成很早就持有，早年股利會被高估。）

## 測試

把 `tests.html` 與 `index.html` 放同一資料夾，用瀏覽器開啟 `tests.html`（或部署後開 `.../tests.html`）。
它以隱藏 iframe 載入 index.html，直接測試其 `window.StockCalc`（`curShares`／`avgCost`／`sharesAt`／`cashDivReceived`），不複製程式碼、不會與主程式脫鉤；畫面會顯示每項通過／失敗與總計。

## 資料來源與限制

- 即時報價：台灣證交所 `mis.twse.com.tw`，純前端無法直連，經公用 CORS proxy（corsproxy / codetabs / allorigins）競速取得，proxy 偶有不穩。
- 股利、股價、股票清單：[FinMind](https://finmindtrade.com/) API（瀏覽器可直連）。免費額度有每小時請求上限（匿名依 IP 約 300 次/時），達上限時稍後再按一次同步即可。
- 股利涵蓋上市與上櫃 ETF／個股；少數冷門標的或極新上市可能查無。
- 本工具僅供個人記錄參考，不構成投資建議。
