# 台股即時報價

單一 HTML 的台股自選股報價頁。輸入股票代號加入清單，自動定時更新，清單保存在瀏覽器。

## 功能

- 輸入代號加入自選（上市如 `2330`、上櫃如 `6488`）
- 顯示成交、漲跌、幅度、開高低、昨收、成交量
- 自選清單存於瀏覽器 `localStorage`，重整不消失
- 可選更新間隔（10／30／60 秒或不自動）

## 使用

直接用瀏覽器開啟 `index.html` 即可。

部署到 GitHub Pages：repo → Settings → Pages → Source 選 `main` / root，
之後可由 `https://你的帳號.github.io/tw-stock/` 開啟。

## 資料來源與限制

- 來源：台灣證交所即時報價 API（`mis.twse.com.tw`）。
- 純前端網頁無法直連（CORS），透過公用 proxy（allorigins／corsproxy）轉接；
  proxy 為免費公共服務，偶爾不穩，失敗時頁面會顯示「更新失敗」並重試下一個 proxy。
  若需穩定，請自架後端代理。
- 非交易時段無成交價，改顯示昨收或最佳買賣價；盤中資料約有數秒延遲，非毫秒級即時。
